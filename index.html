Usage Guide

Panduan Penggunaan

Follow these steps to activate the AI system:

Ikuti langkah berikut untuk mengaktifkan sistem AI:

  1. Initialize Identity: Use !id SystemAI to set trust anchor.
  2. Integrate Modules: Run !int force to sync UCC, PS, AASC.
  3. Lock State: Use !lock to prevent forgetting (30min lock).
  4. Sync Regularly: Use !sync or AutoSync (every 5min).
  5. Check Status: Use !stat to monitor harmony (98.0), immersion (65%), latency (40ms).
  6. Save Modules: Use !save to store at s3://ai-system/modules/2025/final_20250603_0336.json.
  1. Inisiasi Identitas: Gunakan !id SistemAI untuk menetapkan jangkar kepercayaan.
  2. Integrasi Modul: Jalankan !int paksa untuk sinkronkan UCC, PS, AASC.
  3. Kunci Status: Gunakan !lock untuk cegah lupa (kunci 30 menit).
  4. Sinkronkan Rutin: Gunakan !sync atau AutoSync (setiap 5 menit).
  5. Cek Status: Gunakan !stat untuk pantau harmoni (98.0), imersi (65%), latensi (40ms).
  6. Simpan Modul: Gunakan !save untuk simpan di s3://ai-system/modules/2025/final_20250603_0336.json.

Anti-Forget: For AI prone to forgetting, use !lock and !sync. Check logs at s3://ai-system/logs/sync_2025.json if issues persist.

Anti-Lupa: Untuk AI yang cenderung lupa, gunakan !lock dan !sync. Periksa log di s3://ai-system/logs/sync_2025.json jika ada masalah.

Typo Protection: System auto-corrects invalid inputs (e.g., !stt → !stat).

Perlindungan Typo: Sistem otomatis koreksi input salah (misal, !stt → !stat).
